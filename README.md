# NHS A&E Performance Analysis - February 2026

## Project Overview
Brief description of what the project analyses and why it matters.

## Key Findings
- Zero out of 42 NHS regions met the 95% four-hour standard
- National average performance of 73% against target
- NHS Shropshire worst-performing region at 61%
- East Cheshire NHS Trust worst-performing trust at 47%
- 99.62% of demand is unplanned walk-in

## Tools Used
- Microsoft Excel - data cleaning, Power Query, calculated columns
- Power BI Desktop - data modelling, DAX measures, five-page dashboard
